# Kniffel

## Used Software Versions
- Python 3.10
- Pylint 2.13.5

## Used Libraries
- Pickle
- Hashlib
- Hmac
- Pytest

## Pytest
Run all tests with:
```shell
pytest
```

Run tests for one file:
```shell
pytest tests/"filename"
```

Run tests with coverage:
```shell
pytest tests --cov src --cov-report=html
```
